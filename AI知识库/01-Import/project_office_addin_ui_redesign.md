# Office Add-in UI 重设计方案 v1.0

## 设计理念
「AI是文档的精确助手，不是侧边栏里的聊天机器人」

## 核心决策

| 维度 | 决策 |
|------|------|
| 布局 | B+C混合：智能浮动 + 分屏协作 |
| AI面板位置 | 右侧默认 + 可拖拽（280-50%窗口宽度） |
| 触发方式 | 点击浮动按钮 + 智能检测（公式/表格/长文本自动弹出） |
| 设计风格 | 暖Editorial + 毛玻璃 |
| 三端差异化 | 颜色 + 功能面板 + 布局微调 |
| AI交互 | 插入文档 + 复制 + 一键执行 |

## 核心交互流

```
选中文本 → 浮动按钮出现
    ↓
点击浮动按钮 / 智能检测触发 → AI气泡弹出
    ↓
需要深度对话？ → 点击"展开" → 分屏模式（右侧AI面板）
    ↓
完成对话？ → 点击"收起" → 回到气泡模式
    ↓
高级用户？ → Ctrl+K → 命令面板
```

## 智能检测触发规则

| 应用 | 触发条件 |
|------|----------|
| Excel | 选中≥3行连续数值 / 存在错误公式 / ≥200字符长文本 |
| Word | 选中≥100字文本 / 存在拼写语法错误 / 插入引用内容 |
| PPT | 选中≥3页幻灯片 / 存在排版混乱 / 未加备注页面 |

## 三端差异化

| 应用 | 颜色 | 布局 | 功能面板 | 快捷键 |
|------|------|------|----------|--------|
| Excel | 绿 #217346 | 紧凑网格（4px/8px间距） | 公式调试/图表/数据/清洗 | Alt+Shift+A |
| Word | 蓝 #2B579A | 宽松排版（8px/16px间距） | 润色/大纲/翻译/校对 | Ctrl+Shift+E |
| PPT | 红 #D24726 | 卡片化（12px/24px间距） | 幻灯片/结构/备注/美化 | Ctrl+Shift+M |

## 设计规范

| 元素 | 规范 |
|------|------|
| 字体 | Playfair Display + Source Sans 3 + 中文fallback |
| 圆角 | 12px(卡片)/8px(按钮)/20px(气泡)，hover-2px |
| 毛玻璃 | backdrop-filter:blur(12px)，支持浅色/深色/高对比度 |
| 动效 | 气泡300ms弹性，分屏200ms ease-in-out，可关闭 |
| 间距 | 8px基准网格，三端差异化 |

## 防打扰逻辑

- 全局开关：可关闭智能检测
- 自动识别演示/共享屏幕模式 → 免打扰
- 气泡3秒无交互自动收起
- 点击空白处/ESC自动消失
- 浮动按钮边缘吸附，不遮挡内容

## 隐私合规

- 智能检测仅本地运行，不上传文档数据
- 仅用户点击执行时上传最小范围数据
- 支持本地模型部署
- 气泡旁显示「本地检测」锁标识

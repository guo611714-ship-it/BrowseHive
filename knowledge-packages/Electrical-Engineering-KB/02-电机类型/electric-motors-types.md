# 电机类型分类

## 概述

[[电机]]是将电能转换为机械能的设备，按工作原理和结构可分为多种类型。

## 主要分类

### 按电流类型

#### 直流电机 (DC Motor)

**特点**:
- 使用直流电源
- 调速性能好
- 启动转矩大

**结构**:
- [[电枢]] (转子)
- [[定子]] (产生磁场)
- [[换向器]] + [[电刷]]

**应用**: 电动车窗、电动工具、起重机

#### 交流电机 (AC Motor)

交流电机又分为[[异步电机]]和[[同步电机]]两大类。

---

### 异步电机 / 感应电机 (Induction Motor)

**工作原理**:
定子产生旋转磁场，转子感应电流，电磁力驱动转子旋转。转子转速**低于**同步转速（有滑差）。

**类型**:
1. **鼠笼式异步电机** (Squirrel Cage IM)
   - 转子: 导电条短路环
   - 优点: 结构简单、成本低、维护少
   - 应用: 工业泵、风机、压缩机

2. **绕线式异步电机** (Wound Rotor IM)
   - 转子: 三相绕组，可外接电阻
   - 优点: 启动转矩大、可调速
   - 应用: 起重机、卷扬机

**关键参数**:
- 转差率: s = (n_s - n) / n_s
- 效率: 85%-95%
- 功率因数: 0.7-0.9

---

### 同步电机 (Synchronous Motor)

**工作原理**:
转子转速**等于**定子旋转磁场转速（同步转速）。

**结构**:
- **励磁方式**:
  - 他励: 独立直流电源
  - 永磁: 使用稀土永磁体 (钕铁硼)
- **转子**: 直流励磁绕组或永磁体
- **定子**: 三相绕组

**特点**:
- 功率因数可调 (可超前/滞后)
- 效率高 (95%-98%)
- 结构复杂、价格高

**应用**:
- 大型水泵、压缩机
- 发电机 (同步发电机)
- 精密转速场合

---

### 步进电机 (Stepper Motor)

**原理**:
将脉冲信号转换为精确的角度步进。每输入一个脉冲，转子转过一个固定角度（步距角）。

**类型**:
1. **反应式** (VR) - 定子齿吸引转子齿
2. **永磁式** (PM) - 转子为永磁体
3. **混合式** (HB) - 结合两者优点

**优点**:
- 开环控制 (无需反馈)
- 精确定位
- 保持转矩 (断电后保持位置)

**应用**:
- CNC机床
- 3D打印机
- 办公自动化设备

**相关**: [[伺服电机]] (闭环控制)

---

### 伺服电机 (Servo Motor)

**特点**:
- 高精度位置/速度/转矩控制
- 闭环系统 (编码器反馈)
- 快速响应

**组成**:
- 电机本体 (通常为永磁同步电机)
- 位置/速度传感器 (编码器、旋变)
- 驱动器 (伺服放大器)

**应用**:
- 机器人关节
- 半导体设备
- 工业自动化

**对比**: 伺服 vs 步进 - 伺服用在高速、高精度、负载变化大场合

---

### 特殊类型电机

#### 无刷直流电机 (BLDC)

- 无电刷和换向器
- 电子换向
- 效率高、寿命长
- 应用: 无人机、电动车、精密仪器

#### 磁阻电机 (Reluctance Motor)

利用磁阻最小原理:
- 开关磁阻电机 (SRM)
- 磁阻同步电机

特点: 结构简单、可靠性高

#### 直线电机 (Linear Motor)

将电能直接转换为直线运动:
-  Induction Linear Motor
-  PM Linear Motor

应用: 磁悬浮列车、直线执行器

## 选择指南

| 应用需求 | 推荐类型 | 理由 |
|---------|---------|------|
| 通用工业 | [[鼠笼式异步电机]] | 成本低、维护少 |
| 调速要求高 | [[异步电机]] + [[变频器]] | 高效调速 |
| 精密同步 | [[同步电机]] | 转速准确、可调功率因数 |
| 精确定位 | [[步进电机]] | 开环定位 |
| 高性能运动 | [[伺服电机]] | 闭环高动态响应 |
| 直流电源系统 | [[直流电机]] | 直接使用直流电 |
| 低成本/小功率 | BLDC | 高效、免维护 |

## 性能参数对比

| 参数 | 异步电机 | 同步电机 | 直流电机 |
|------|---------|---------|---------|
| 效率 | 85-95% | 95-98% | 80-90% |
| 功率因数 | 0.7-0.9 (滞后) | 可调 | 取决于励磁 |
| 调速范围 | 宽 (需变频) | 宽 | 极宽 |
| 维护成本 | 低 (无刷) | 中 (有励磁) | 高 (有电刷) |
| 成本 | 低 | 高 | 中 |
| 启动转矩 | 中 | 低 (需辅助) | 高 |

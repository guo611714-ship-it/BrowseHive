# 变频驱动器 (VFD) 技术

## 概述

[[变频器]] (Variable Frequency Drive, VFD) 是通过改变频率来控制[[交流电机]]转速的电力电子装置，是实现电机节能调速的核心设备。

## 基本工作原理

```
交流电源 (50/60Hz)
    ↓
整流器 (AC→DC)
    ↓
直流中间电路 (DC Bus)
    ↓
逆变器 (DC→AC, 可变频率/电压)
    ↓
[[交流电机]] (转速 n = 60f/p)
```

其中:
- f: 电源频率 (Hz)
- p: 电机极对数
- n: 同步转速 (rpm)

---

## 主要拓扑结构

### 1. 电压源型变频器 (VSI - Voltage Source Inverter)

- **直流侧**: 电压基本稳定 (大电容滤波)
- **应用**: 通用变频器、风机水泵
- **输出**: 近似正弦波PWM

### 2. 电流源型变频器 (CSI - Current Source Inverter)

- **直流侧**: 电流基本稳定 (大电感滤波)
- **应用**: 大功率 (>1MW) 场合
- **特性**: 四象限运行、 robust

### 3. 矩阵式变频器 (Matrix Converter)

- 直接AC-AC变换，无中间直流环节
- 优点: 功率因数高、输入输出特性好
- 缺点: 控制复杂、成本高

---

## 核心部件

### 整流器 (Rectifier)

将交流转换为直流:

**类型**:
- **不可控整流** (二极管) - 成本低，不能回馈制动
- **可控整流** (晶闸管/SCR) - 可四象限，单向不能 regenerative
- **PWM整流** (IGBT) - 双向、高功率因数

### 直流中间电路 (DC Link)

**作用**: 储能、滤波、提供稳定直流电压

**元件**:
- **大容量电解电容** (VSI型) - 吸收脉动
- **大电感** (CSI型) - 平滑电流

**缺点**: 电容是VFD寿命短板 (通常10年)

### 逆变器 (Inverter)

将直流转换为交流:

**开关器件**:
- **IGBT** - 通用 (600V-3300V, <10kHz)
- **SiC MOSFET** - 高效高频 (开关损耗低)
- **GTO** - 大功率 (逐渐淘汰)

**调制技术**:
- **PWM** (Pulse Width Modulation) - 最常用
  - SPWM (正弦脉宽调制)
  - SVPWM (空间矢量PWM) - 电压利用率高15%
- **多电平PWM** - 高压大功率
  - NPC (三电平)
  - Flying capacitor
  - Cascaded H-bridge

## 控制策略

### V/f 控制 (标量控制)

最简单控制:
- 保持电压/频率比恒定 (U/f = const)
- 适用于: 风机、水泵等"平方律负载"
- 优点: 简单、便宜
- 缺点: 动态性能差

### 矢量控制 (Vector Control / FOC)

**目标**: 像控制直流电机一样控制交流电机

**核心**:
1. Clarke变换: αβ坐标
2. Park变换: dq坐标 (随转子旋转)
3. 解耦控制: d轴电流→磁链, q轴电流→转矩

**优点**:
- 四象限运行
- 快速转矩响应 (<5ms)
- 高精度调速

**需**: 速度/位置传感器 或 无传感器算法

### 直接转矩控制 (DTC - Direct Torque Control)

**特点**:
- 直接控制转矩和磁链
- 无需PWM调制
- 转矩响应极快 (<2ms)
- 开关频率可变

**缺点**:
- 转矩脉动较大
- 低速性能受电机参数影响

### 预测控制 (MPC)

未来趋势:
- 基于电机模型预测下一状态
- 优化开关选择
- 性能优异，计算量大

## 应用分类

### 通用变频器 (General Purpose)

- 功率: 0.4kW - 500kW
- 应用: 传送带、风机、水泵
- 功能: V/f控制，基本PLC

### 高性能变频器 (High Performance)

- 功率: 0.4kW - 数MW
- 应用: 电梯、起重机、机床
- 功能: 矢量控制、位置控制、高精度

### 专用变频器

- **电梯专用**: 抱闸控制、平层精准
- **注塑机专用**: 多缸同步
- **纺织专用**: 卷绕控制
- **轨道交通**: 牵引变流器 (IGBT串联)

## 节能原理

### 风机水泵变频节能

风机/水泵流量Q ∝ n (转速)
功率P ∝ n³ 或 Q³

**举例**:
- 转速降为80% → 流量降为80% → 功率降为51%
- **节能率达49%**

**投资回收**: 通常<2年

---

## 技术趋势

### 1. 更高效率

- **SiC/GaN器件** - 开关损耗降低50%+
- **更高开关频率** - 减小滤波器体积
- **效率目标**: 98%+ (目前95-97%)

### 2. 更智能

- **内置PLC、安全功能**
- **云连接** - 远程监控、预测性维护
- **自适应参数** - 电机参数自学习

### 3. 更紧凑

- **集成电机驱动** - 电机+驱动器一体化
- **模块化设计** - 降低备件库存
- **散热技术进步** - 更小体积

---

## 安装与维护注意事项

### 安装

1. **散热**: 保证通风，避免高温环境 (>40℃降额)
2. **EMI**: 输入/输出加滤波器，远离敏感设备
3. **制动**: 制动电阻/单元选型正确
4. **电缆**: 输出电缆长度有限制 (长线反射)

### 维护

- **定期**: 清洁散热风扇/风道
- **电容**: 10年左右更换直流母线电容
- **监控**: 关注运行温度、电流、故障次数

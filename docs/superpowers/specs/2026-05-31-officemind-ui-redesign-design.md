# OfficeMind UI 全面重构设计规格

**日期**: 2026-05-31
**版本**: 2.0
**状态**: 待审批

## 1. 概述

基于三张截图的全面分析，对 OfficeMind (Claude Excel Assistant) 进行 UI 全面重构。覆盖交互、视觉、布局、体验四大维度，分三阶段执行。

## 2. 执行阶段

| 阶段 | 内容 | 预估工时 |
|------|------|----------|
| A+B | 交互修复 + 视觉重构（7项） | 5-7天 |
| C | 信息层级与布局重设计 | 3-4天 |
| D | 用户体验与易用性优化 | 2-3天 |

## 3. A+B 阶段设计决策

### 3.1 AI 气泡双模式切换

**决策**: 方案 C — 侧边栏默认 + 浮动面板可切换

**侧边栏模式（默认）**:
- 保持当前侧边栏布局
- 右上角添加"弹出"按钮（图标：方框+向外箭头）
- 点击后内容无缝迁移到浮动面板，滚动位置和输入内容完全保留
- 弹出后侧边栏显示"点击重新打开 OfficeMind"提示

**浮动面板模式**:
- 支持全屏幕自由拖动
- 支持调整大小（最小宽度 320px，最小高度 400px）
- 支持边缘吸附（拖动到屏幕边缘时自动对齐）
- 支持最小化、最大化/还原
- 标题栏：OfficeMind logo + 名称 | 最小化 | 最大化 | 停靠按钮 | 关闭
- 停靠按钮（图标：方框+向内箭头）将内容迁回侧边栏
- 记住用户上次窗口大小和位置
- 始终置顶但不抢焦点

**快捷键**:
- `Ctrl+Shift+O`: 打开/关闭 AI 助手
- `Ctrl+Shift+P`: 侧边栏↔浮动面板切换
- `Ctrl+Enter`: 发送消息
- `Esc`: 关闭浮动面板

**多显示器**: 允许拖到任意显示器，为 Excel/Word/PPT 分别记住位置

### 3.2 操作按钮三层体系

**决策**: A + C + B 三层全覆盖

**底层 — 底部永久操作栏**:
- 插入到文档（主色图标+文字）、复制（仅图标）、重试（仅图标）
- 操作栏与回复气泡之间 8px 间距
- 回复超过 300 字时，操作栏固定在聊天窗口底部

**中层 — 选中文字跟随工具条**:
- 选中回复中任意文字时，选区上方 10px 显示迷你工具条
- 按钮：插入选中内容、复制选中内容、翻译选中内容
- 跟随选区移动，取消选区时自动消失
- 右键点击时隐藏，避免与系统菜单重叠

**顶层 — 鼠标悬停悬浮工具条**:
- 鼠标悬停在回复上时，右上角显示高级操作
- 包含：重新生成、点赞/踩、分享、导出为 PDF
- 半透明显示，鼠标移到工具条上时变不透明
- 延迟 300ms 显示，避免快速划过误触发

**插入反馈**:
- 插入成功后，文档中插入位置显示短暂高亮动画
- 聊天窗口显示"已插入到文档"提示
- 插入后 5 秒内显示"撤销插入"按钮
- 保留 AI 回复中的格式（加粗、列表、表格）

**插入位置选择**:
- 点击"插入到文档"后提供下拉选项：
  - 插入到光标位置（默认）
  - 插入到文档末尾

### 3.3 弹窗完全移除

**决策**: 方案 C — 完全移除主动弹窗 + 入口引导优化

**移除内容**:
- 删除"检测到文档内容"阻塞式弹窗
- 删除 AI 浮动按钮的弹出逻辑（改为侧边栏入口）

**入口优化**:
- "分析当前文档"按钮移至侧边栏顶部，"新对话"按钮下方
- 使用主色填充按钮，比其他功能更突出
- 按钮文字："分析当前文档"

**首次使用引导**:
- 第一次打开时，在"分析当前文档"按钮旁显示气泡提示
- 提示文字："点击这里，让 AI 帮你分析整个文档内容"
- 用户点击一次后永久消失

**上下文感知提示（可选）**:
- 打开大 Excel 表格时，输入框占位符显示："试试让 AI 分析当前文档，帮你快速提取关键信息"
- 用户开始输入时占位符自动消失

**非阻塞通知**:
- 所有通知在侧边栏顶部以横幅显示
- 3 秒后自动消失，可手动关闭
- 支持通知历史记录

### 3.5 主题切换文字同步

**决策**: 修复 `theme-switcher.js` 中的文字逻辑

**当前问题**: 浅色模式下显示"切换到暗色模式"（正确），但截图中界面状态与文字不一致

### 3.6 深色模式调色板重构

**决策**: 方案 C — 全面重构 + WCAG AAA 合规

**背景色系统**:
```css
--dark-bg-primary: #1E1E1E;    /* 主背景（卡片/侧边栏） */
--dark-bg-secondary: #2D2D2D;  /* 次背景（输入框/下拉菜单） */
--dark-bg-tertiary: #383838;   /* 三级背景（悬浮/选中） */
```

**文字色系统（WCAG 对比度）**:
```css
--dark-text-primary: #FFFFFF;   /* 14.1:1 → AAA */
--dark-text-secondary: #B3B3B3; /* 8.6:1 → AAA */
--dark-text-disabled: #7A7A7A;  /* 4.6:1 → AA */
```

**边框与交互**:
```css
--dark-border: #3A3A3A;         /* 4.7:1 */
--dark-primary: #10B981;        /* 7.2:1 → AAA */
```

**参照标准**: Google Material Dark (#121212 基调) + Office Dark Theme

### 3.7 图标差异化

**决策**: 方案 A — 每功能独立语义化线性图标

**图标分配**:
| 功能 | 图标描述 |
|------|----------|
| 总结数据 | 柱状图/摘要图标 |
| 生成透视表 | 表格网格图标 |
| 分析趋势 | 折线图图标 |
| 查找异常 | 放大镜+警告图标 |
| 调试公式 | 代码/bug图标 |
| 构建模型 | 流程图/架构图标 |
| 工作表 | 网格/工作表图标 |
| 公式依赖 | 节点连接图图标 |
| 连接器 | 插头/链接图标 |

**风格规范**:
- 统一线性风格，stroke-width: 1.5-2px
- 尺寸：16x16（按钮内）/ 24x24（侧边栏）
- 悬停时：轻微缩放 + 颜色过渡动画
- 点击时：反馈动效

### 3.8 品牌名称统一

**决策**: 全局统一为 "OfficeMind"

**修改点**:
- `taskpane.html` title: "OfficeMind Excel Assistant" → "OfficeMind"
- 欢迎文字: "OfficeMind 智能助理"（保持）
- 移除所有 "Claude Excel Assistant" 引用
- 技术标识: "Powered by Claude"（可选，放在设置页）

## 4. C 阶段设计决策（待细化）

- 信息层级重排：主要操作主色填充，次要操作描边按钮
- 功能分组：数据处理 / 内容生成 / 辅助工具 三组
- 侧边栏布局优化：搜索框 + 分组 + 可折叠
- 空间利用率：增加默认宽度，减少留白

- 操作说明：每个功能添加简短提示

- 首次引导：分步高亮引导流程

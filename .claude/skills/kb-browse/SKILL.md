# AI知识库浏览

## 用法
```
/kb-browse
```

## 功能
显示知识库的完整结构，包括Wiki页面和原始文档。

## 执行步骤
1. 扫描 `AI知识库/` 目录结构
2. 统计各分类文档数量
3. 显示最近导入的文档
4. 显示索引和知识图谱状态

## 输出格式
```
📚 AI知识库结构
├── wiki/        (X 个页面)
├── 01-Import/   (X 个文档)
├── 02-Notes/    (X 个笔记)
└── 03-Index/    (索引状态)

📊 统计: 总文档 X | 概念 X | 实体 X
```

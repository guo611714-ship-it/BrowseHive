# 技能库管理

## 功能
- 搜索现有技能包
- 安装新技能
- 列出已安装技能

## 使用
```
load_skill clawhub "搜索天气技能"
```

## 示例回复
```
找到以下相关技能:
1. weather - 天气查询
2. forecast - 天气预报
使用: load_skill weather "查询北京天气"
```

# Agent Team 图标生成提示词

## 风格统一要求
- 扁平化设计，简约线条
- 圆形构图，适合头像使用
- 明朝古风元素，非现代Q版
- 单一主色调，背景透明或浅色
- 32x32 至 128x128 像素

## 使用建议

1. **统一尺寸**: 生成 512x512 像素，导出为 128x128
2. **格式**: PNG 透明背景
3. **一致性**: 所有提示词都包含 "flat design, minimalist, circular composition" 保持风格统一
4. **差异化**: 每个角色有独特的配饰和主色调，便于区分
5. **命名规则**: `agent-{id}.png`，如 `agent-xiaohuangmen.png`

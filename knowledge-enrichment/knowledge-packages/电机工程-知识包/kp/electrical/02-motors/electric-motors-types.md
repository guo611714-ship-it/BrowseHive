# 电机类型分类

## 概述

电机是将电能转换为机械能的设备，按工作原理和结构可分为多种类型。

## 主要分类

### 直流电机 (DC Motor)

- 使用直流电源
- 调速性能好
- 启动转矩大
- 含[[换向器]]和[[电刷]]

### 异步电机 / 感应电机

**鼠笼式**: 结构简单，成本低，维护少
**绕线式**: 启动转矩大，可调速

特点: 转子转速低于同步转速

### 同步电机

转子转速等于同步转速。
- 功率因数可调
- 效率高 (95-98%)
- 应用: 大型水泵、发电机

### 步进电机

将脉冲信号转换为精确角度步进。
- 开环控制
- 精确定位

### 伺服电机

- 高精度位置/速度控制
- 闭环系统
- 快速响应

## 选择指南

| 应用 | 推荐类型 |
|------|---------|
| 通用工业 | [[鼠笼式异步电机]] |
| 调速要求高 | [[异步电机]] + [[变频器]] |
| 精密同步 | [[同步电机]] |
| 精确定位 | [[步进电机]] |
| 高性能运动 | [[伺服电机]] |

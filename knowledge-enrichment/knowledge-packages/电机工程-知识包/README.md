## 🚀 使用方法

1. 将本文件夹复制到LLM Wiki项目的 `Import/` 目录
2. 打开LLM Wiki，点击Import或等待自动导入
3. 生成的Wiki页面会自动出现在 `wiki/` 文件夹
4. 在Obsidian中打开该 `wiki/` 文件夹即可浏览知识图谱

## 📖 文档格式

所有文档采用统一的YAML frontmatter格式，包含：
- title: 文档标题
- source: 来源标识
- tags: 标签
- entities: 实体列表（用于双向链接）
- created: 创建日期

## 🔗 关联性

文档已预先建立双链关系，例如：
- [[电动机]] 链接到 [[异步电机]]
- [[电力系统]] 链接到 [[输电网络]]
- [[变频器]] 链接到 [[电机控制]]

在Obsidian的Graph View中可以看到完整的知识网络。

## 📊 知识图谱

导入后，建议：
1. 在Obsidian中按 `Ctrl+G` 打开Graph View
2. 使用筛选器显示特定标签（如 `#电机`）
3. 识别核心枢纽节点（连接最多的文档）

## 🔄 更新与扩展

如需添加新内容：
1. 创建新文档，遵循相同格式
2. 在frontmatter中添加适当的 `entities`
3. 在文档中使用 `[[内部链接]]` 引用已有页面
4. 重新运行LLM Wiki处理以更新图谱
